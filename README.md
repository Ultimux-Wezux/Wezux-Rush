Project: WEZUX HYBRID
WEZUX HYBRID is a high-stakes, neon-styled reaction and speed-clicking arcade challenge. The game evaluates your physical performance through two distinct phases: a reaction time test followed by a rapid-fire clicking sequence.  Gameplay OverviewInitialization: The system idles until the user triggers the protocol.  Reaction Phase: A "waiting" period precedes the "CLICK!" signal; clicking during the waiting period results in a false start.  Performance Phase: After reacting, the user must click as rapidly as possible within a 5-second window.  Ranking Hierarchy: A Performance Index is calculated based on the formula $(\frac{\text{Clicks}}{\text{Reaction Time}} \times 1000)$. Ranks are determined by this index:  HUMAN < CYBER < NIGHT CORE < DAY CORE < ARTIFICIAL INTELLIGENCE < WEZUX.  FeaturesModern UI: Built with a sleek, high-contrast, and minimalist glassmorphism design.  Dynamic Feedback: Includes a real-time progress bar during the clicking phase and responsive visual states for user input.  System Integrity: Includes robust logic to prevent cheating and handle errors with professional re-initialization (REBOOT) protocols.
